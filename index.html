Learning Unit 01: Grundlagen von HTML

Kurze Zusammenfassung:

  • HTML ist die Markup-Sprache zur Strukturierung von Webseiten.
  • Elemente wie <div>, <p>, und <a> sind grundlegend.

Beispiel:

<!DOCTYPE html>
<html>
<head>
<title>Beispiel</title>
</head>
<body>
<p>Dies ist ein Absatz.</p>
</body>
</html>

Learning Unit 02: Einführung in CSS

Kurze Zusammenfassung:

  • CSS steuert das Aussehen der Webseite.
  • Wichtige Konzepte sind Selektoren, Eigenschaften und Werte.

Beispiel:

<style>
body { background-color: lightblue; }
h1 { color: white; text-align: center; }
p { font-family: verdana; font-size: 20px; }
</style>

Learning Unit 03: JavaScript Grundlagen

Kurze Zusammenfassung:

  • JavaScript ist eine Programmiersprache für dynamische Webseiten.
  • Verwendung von document und window für DOM-Manipulation.

Beispiel:

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>
